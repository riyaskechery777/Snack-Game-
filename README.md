A modern, responsive implementation of the classic Snake game with smooth animations, score tracking, and an intuitive interface.

## ✨ Features

- **Smooth Controls** - Arrow keys or WASD support
- **Score Tracking** - Local storage for high scores
- **Pause/Resume** - Press SPACE or click button
- **Visual Effects** - Gradient snake, glowing food, and smooth animations
- **Responsive Design** - Plays on desktop and tablets
- **Game States** - Start, Pause, Reset functionality

## 🎯 How to Play

1. Click **Start Game** to begin
2. Use **Arrow Keys** or **WASD** to control the snake
3. Eat the red food squares to grow and increase your score
4. Avoid hitting the walls or your own tail
5. Press **SPACE** or click **Pause** to pause/resume
6. Try to beat your high score!

## 🛠️ Technologies Used

- HTML5 Canvas
- CSS3 (Gradients, Animations, Flexbox)
- Vanilla JavaScript (ES6+)
- Local Storage API
