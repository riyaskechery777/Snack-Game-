## 🎯 How to Play

1. **Start the game** — Click the "START GAME" button
2. **Control the snake** — Eat the red apples to grow longer
3. **Avoid collisions** — Don't hit the walls or your own tail
4. **Score points** — Each apple = +1 point
5. **Beat your high score** — Try to get the highest score possible!

## 🚀 Deployment Instructions

### Option 1: GitHub Pages (Free Hosting)

1. Create a new repository on GitHub (e.g., `snake-game`)
2. Upload the `index.html` file to the repository
3. Go to **Settings → Pages**
4. Under "Branch", select `main` and `/root` folder
5. Click **Save**
6. Your game will be live at:  
   `https://YOUR-USERNAME.github.io/snake-game/`

### Option 2: Local Play

1. Download the `index.html` file
2. Double-click to open in any modern browser
3. Start playing immediately — no installation needed!

### Option 3: Share with Friends

- Email the `index.html` file
- Host on any static file server
- Use services like Netlify, Vercel, or Cloudflare Pages
